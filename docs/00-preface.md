随着网络时代的深度融入生活，人们的生活方式也在悄悄发生着变化。

阅读是很多人的休闲娱乐甚至是学习和生活方式。曾几何时，阅读就是指人们对纸质媒介（图书、期刊、报纸等）的浏览，逐渐变成了现在对数字媒介（手机、平板、电脑等）的浏览了。随着数字媒体的使用增多，如何编辑出版发行数字文献的需求与日俱增。

数字文献不单单是对于传统文献的扫描数字化，更多的应该是从文献的形成开始就是以数字形式诞生的。例如一篇博客文章，一个论坛帖子，一个网络评论等等。那么对等于传统媒介的大部头的图书的数字文献是什么样子的呢？实际上并没有定论。如果一个以某个专题为主形成的一系列的文章构成的网站，如果再以一定的机构进行组织就可以是一部图书。

Jekyll 是一个可以将简单格式（通常是 Markdown 格式）的内容转换为网页，并以静态网站形式呈现的软件。
